# Ideas

Parking lot for thoughts not scoped into current work. Better to capture a half-formed idea here than lose it.

## Rules

- One file per idea or theme.
- Files can be messy — this is not a polished doc.
- Add a one-line summary at the top of each file.
- When an idea graduates into active work, either:
  - Move it into a real doc (architecture, ADR, etc.), or
  - Delete it after it's clearly obsolete.
- Don't quietly rewrite other people's ideas — add your own file or comment at the bottom of the existing one.
